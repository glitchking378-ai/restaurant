## System Overview

This project simulates a restaurant using Cloudflare Workers.

### Roles
- **Waiter (A Worker)** — Handles customer orders
- **Chef (B Worker)** — Prepares food and validates menu items

### Flow
1. Table places order
2. Waiter receives order
3. Waiter sends order to Chef
4. Chef validates and cooks
5. Food is returned to Waiter
6. Waiter serves the table
